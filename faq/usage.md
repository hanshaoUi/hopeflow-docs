# 常见问题：功能使用

## 工具执行后没有任何反应？

最常见原因是**没有正确选中对象**。请依次检查：

1. ✅ 已选中至少一个对象（对齐类工具需要选中多个）
2. ✅ 图层没有被锁定或隐藏
3. ✅ 文档不是只读模式（检查标题栏是否有锁定图标）
4. ✅ 没有处于文字编辑状态（按 <kbd>Esc</kbd> 退出文字编辑）

如果以上都正常，请在 [GitHub Issues](https://github.com/hanshaoUi/hopeflow-docs/issues) 提交反馈，并附上：
- Adobe Illustrator 版本号
- 操作系统版本
- 操作的对象类型（路径、文字、图像等）

## 排料优化工具运行很久没有结果？

排料优化使用迭代算法，对象数量越多耗时越长。建议：

- 单次操作对象数量控制在 **200 个以内**
- 复杂路径可先用"简化路径"工具降低节点数
- 在设置中降低**迭代轮次**可获得更快（精度略低）的结果
- 超过 60 秒可按 <kbd>Esc</kbd> 中断计算

::: tip 提高排料速度的技巧
将复杂图形先转为矩形边界框参与排料，完成后再替换回原图形，速度可提升 3-5 倍。
:::

## 批量导出时文件名乱码？

这是 Windows 中文路径编码问题，解决方法：

1. 将导出目标路径改为**纯英文路径**（如 `C:\export\`）
2. 或将画板名称改为英文或数字后再导出

::: info v3.1.2 已修复
v3.1.2 修复了路径超过 260 个字符时导致的导出失败问题。请确保已更新到最新版本。
:::

## 颜色工具修改颜色后，文字颜色没有一起改变？

颜色工具默认只处理**填充色**和**描边色**，文字对象的字符颜色需要进入文字编辑状态才能批量修改。

建议使用"文本处理 → 批量替换颜色"工具，该工具专门处理混排文字中的颜色替换。

## 对齐工具相对于哪个对象对齐？

默认以**最后选中的对象**为基准（即关键对象）。如果需要相对于画板或特定对象：

1. 在工具面板顶部切换对齐基准
2. 或先锁定基准对象，再全选进行对齐

## 工具面板空白或加载失败？

尝试以下步骤：

1. 在 Illustrator 中选择 **窗口 → 扩展** → 关闭 HopeFlow Toolbox
2. 清除 CEP 缓存：

::: code-group

```text [Windows 缓存路径]
C:\Users\<用户名>\AppData\Local\Temp\cep_cache\
```

```text [macOS 缓存路径]
~/Library/Caches/CSXS/
```

:::

3. 重新打开面板
