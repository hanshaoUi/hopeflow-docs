# 更新日志

## v3.1.401 <Badge type="tip" text="最新 Release" />

**发布日期：** 2026-04-29

### 优化

- 优化 **大尺寸导出**：分片导出后自动拼接为单张完整图片，不再只保留分片图。
- 大尺寸导出分片改用纯英文临时目录和临时文件名，避免中文路径在 CEP / ExtendScript 环境中转码导致拼接失败。
- 拼接流程增加重叠裁切逻辑，减少分片边缘抗锯齿、裁切误差造成的细缝。
- 拼接后的 JPEG / PNG 会写回导出 DPI 元数据，避免 150 DPI 等设置在合并后显示为 72 DPI。
- 大尺寸导出拼接增加多方案回退：ImageMagick、Python PIL 不可用或失败时继续尝试 Canvas 拼接。

### 修复

- 修复 PNG / JPEG 因 Illustrator 缩放上限导致拼接坐标按目标尺寸拉开、画面出现空洞的问题。
- 修复 Python PIL 未安装时拼接流程误判可用并直接失败的问题。
- 修复大尺寸导出单图合并结果文件名和目标目录处理不稳定的问题。

## v3.1.4

**发布日期：** 2026-04-24

### 新增

- 新增 **导出画板尺寸表格** 面板。
  - 支持按画板或按尺寸分组。
  - 支持单位选择和 1:X 比例换算。
  - 支持 XLSX / CSV 导出。
  - XLSX 支持嵌入画板图示。
  - 支持成品面积、单价、金额列。
  - 金额列可根据 `单价 × 成品面积` 自动计算。
- 新增 Windows 图形安装器 `HopeFlow-Installer.bat`。
- 新增 Windows 发布包命令 `npm run package:windows`。

### 优化

- 画板尺寸表导出支持文档目录、桌面和弹窗选择目录。
- 导出表格前自动保存当前 AI 文档。
- 画板图示以临时 PNG 形式嵌入 XLSX，不再使用 HTML。
- 图示插入 Excel 时保持原始比例，避免拉伸变形。
- 按画板创建矩形时，每次创建独立图层存放结果。
- Windows 安装方式改为复制发布包，用户电脑不再需要 Node.js。

### 修复

- 修复 CEP 环境中 ExcelJS `createWriteStream` / `readFile` 不可用导致导出失败的问题。
- 修复部分 Illustrator `PARM` 错误导致画板尺寸表导出中断的问题。
- 修复目标图层不可编辑时“按画板创建矩形”失败的问题。

## v3.1.3

**发布日期：** 2026-04-23

### 新增

- 新增 **拼接切割**，支持纵切 / 横切、固定单片尺寸、搭接内含、比例换算、末片按剩余尺寸生成和分片命名规则。

- 批量缩放与编号相关脚本同步调整。

### 修复

- 修复路径脚本与排料逻辑回归问题。
- 修复拼接切割命名输入框在中文输入法下偶发无法正常输入的问题。
