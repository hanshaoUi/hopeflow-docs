# 介绍

HopeFlow Toolbox 是一款面向 **Adobe Illustrator** 生产场景的开源效率插件，基于 CEP（Common Extensibility Platform）技术开发，直接集成到 Illustrator 面板中运行，重点解决重复操作、规则化处理和批量执行的问题。

## 核心特点

- **64 个已接入工具**，覆盖快捷操作、画板、批量、色彩、文字、导出、测量、路径、排料等 11 个已上线分类
- **另含 1 个素材库入口**，后续将继续补齐本地 AI / PSD / PNG 素材管理能力
- **双平台支持**：Windows 7+ 和 macOS 10.12+
- **Illustrator CS6 至最新版 2025** 全版本兼容
- **MIT 开源协议**，完整源代码公开，永久免费

## 技术架构

| 层 | 技术 |
|---|---|
| 面板 UI | React + TypeScript |
| 脚本执行 | ExtendScript（`.jsx`） |
| 宿主通信 | CSInterface / CEP |
| 工具注册 | `src/registry/*.ts` |
| 构建工具 | Webpack |

## 快速开始

根据你的操作系统选择安装方式：

- [Windows 安装指南](./install-windows)
- [macOS 安装指南](./install-macos)

安装完成后，在 Illustrator 菜单栏选择 **窗口 → 扩展 → HopeFlow Toolbox** 即可打开面板。

## 主分支最新进展

截至 **2026-04-23**，主分支已同步新增和优化以下能力：

- 新增 **拼接切割**，支持纵切 / 横切、固定单片尺寸、搭接内含、缩放比例换算、末片按剩余尺寸生成，以及分片命名规则
- 新增 **随机色卡填充**，可直接按文档色板或输入颜色值对选区随机配色
- 新增并优化 **对象标号编组**，处理编号排序、居中和对象绑定场景
- 同步增强 **面积周长计算**、**路径长度测量**、**尺寸标注** 等测量脚本
