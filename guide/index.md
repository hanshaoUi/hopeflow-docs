# 介绍

HopeFlow Toolbox 是一款面向 **Adobe Illustrator** 生产场景的开源效率插件，基于 CEP 技术开发，直接集成到 Illustrator 面板中运行。

它重点解决：

- 重复操作自动化
- 画板与对象批量处理
- 生产尺寸、面积和报价表导出
- 路径、测量、排版和导出流程提效

## 核心特点

- **Illustrator 面板内运行**：无需切换到外部软件。
- **批量处理优先**：围绕生产场景设计，适合重复、规则明确的任务。
- **脚本化能力丰富**：画板、导出、测量、路径、颜色、文字、排料等功能持续扩展。
- **Windows / macOS 支持**：提供 Windows 图形安装器和 macOS 安装脚本。

## 快速开始

根据系统选择安装方式：

- [Windows 安装指南](./install-windows)
- [macOS 安装指南](./install-macos)

安装完成后，在 Illustrator 菜单栏选择：

```text
窗口 > 扩展 > HopeFlow Toolbox
```
